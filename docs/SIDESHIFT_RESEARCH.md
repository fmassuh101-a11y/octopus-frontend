# Investigación Sideshift 2025-2026

## Métricas Clave
- **500,000+ creadores** registrados
- **1,000+ marcas** activas
- **$10M+ pagados** a creadores
- **90% de roles llenados** en menos de 3 días

## Modelo de Precios (Para Marcas)

| Plan | Precio | Jobs | Hires | Invites |
|------|--------|------|-------|---------|
| Starter | $199/mes | 1 | 1-5 | 30 |
| Growth | $299/mes | 2 | 5-15 | 100 |
| Scale | $999/mes | 3 | Ilimitado | Ilimitado |
| Enterprise | $10k+/mes | Managed | Top 1% creators | Dedicado |

## Features Clave

### Para Marcas
1. **One-Tap Apply** - Sin CVs, aplicación instantánea
2. **Job Boosts** - Visibilidad premium para gigs ($$$)
3. **Analytics en tiempo real** - Views, engagement, conversiones
4. **Creator Management Tools** - Dashboard unificado
5. **Modelo Híbrido** - Self-serve + Managed service
6. **Brief posting** - Post una vez, llega a miles de creadores

### Para Creadores
1. **Instant Payouts** - Pago directo al banco
2. **No followers required** - Solo importa el portfolio
3. **One-tap apply** - Sin CVs ni aplicaciones repetitivas
4. **Analytics Dashboard** - Track earnings & engagement
5. **SideShift Pro** - Acceso prioritario a gigs

## Ganancias de Creadores
- Principiantes: **$100-$300 por video**
- Experimentados: **$500-$2,000+ por campaña**

## Cómo Funciona el Flujo

### Para Marcas:
1. Post brief (descripción del gig)
2. Llega a miles de creadores instantáneamente
3. Revisar aplicantes interesados
4. Ver perfiles de creadores
5. Contratar y gestionar desde dashboard
6. Analytics en tiempo real
7. Pagos automáticos

### Para Creadores:
1. Ver gigs disponibles
2. One-tap apply
3. Ser seleccionado
4. Crear contenido
5. Entregar
6. Recibir pago instantáneo

## Modelo de Negocio
- Suscripción mensual para marcas ($199-$999+)
- Job Boosts para visibilidad extra
- Enterprise: servicio managed completo
- Creadores: FREE (plataforma cobra a marcas)
